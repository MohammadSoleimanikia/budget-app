# Budget Management App (TypeScript + React)

A simple **practice project** built with **TypeScript**, designed to explore fundamental concepts of state management, component structure, and data persistence.  
This application allows users to **create budgets**, **add expenses**, and **track overall spending**, where every expense is associated with a specific budget.

---

## Features

- **Add and manage budgets**  
  Create budgets with custom limits to organize and track your spending.

- **Add expenses to each budget**  
  Every expense belongs to one specific budget for clear and structured expense tracking.

- **Responsive UI**  
  A fully responsive interface optimized for both desktop and mobile devices.

- **Global State Management with React Context**  
  Uses **useContext** to handle global state in a clean and scalable way.

- **Persistent Data with Local Storage**  
  All budgets and expenses are saved and restored automatically using the browser’s Local Storage.

---

## Tech Stack

- **React + TypeScript**
- **Vite**
- **React Context API**
- **Local Storage**
- **Tailwind**
